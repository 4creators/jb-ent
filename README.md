# jb-ent

**The Graph-Based Knowledge Base for Code, accessible via Vector Embeddings.**

Project **jb-ent** is a high-performance code intelligence platform that provides a persistent knowledge graph of functions, classes, and call chains across any OS environment, enriched with semantic vector search capabilities.

## Overview
- **Rust Application Layer:** Safe and modern management of MCP, CLI, and configuration.
- **C11 Core:** Blazing fast structural analysis engine based on tree-sitter.
- **Universal Identity:** Stable project identification using Git root commit hashes.
- **3-Level Config:** Flexible System, User, and Project settings.
